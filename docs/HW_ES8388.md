# HW_ES8388

Tags: #done #hardware #codec

Low Power Stereo Audio Codec used for Microphone ADC input. Driver: [[FW_BSP_ES8388]].

---

## Peripheral Overview

- **I2C Address**: `0x10`.
- **I2S Interface**: Connected to I2S0 on [[HW_ESP32P4]].
- **Role**: ADC (Microphone) input.

## Register Map (Hi-Res 96kHz Config)

| Register | Value | Function |
|----------|-------|----------|
| `MASTERMODE` | 0x80 | Set ES8388 as I2S Master |
| `ADCCONTROL4` | 0x08 | I2S Standard, 24-bit data |
| `ADCCONTROL5` | 0x00 | ADC Fs = MCLK / 256 (96kHz) |
| `DACCONTROL2` | 0x08 | I2S Standard, 24-bit data |
| `DACCONTROL3` | 0x00 | DAC Fs = MCLK / 256 (96kHz) |

## Critical Constraints

### Sample Rate
Must be configured for **96kHz** by setting `MCLK/Fs = 256`. Requires external MCLK (24.576 MHz) from [[HW_ESP32P4]] GPIO20.
